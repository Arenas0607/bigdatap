Funcionamiento del Sistema

1. Captura de Datos

Se realiza la lectura de temperatura y humedad mediante los sensores DHT11 y DHT22. La concentración de gas hidrógeno se detecta con el sensor MQ8, y la hora y fecha actual se obtienen desde el módulo RTC DS3231.

2. Envío de Datos

Si hay una conexión WiFi activa, los datos se envían directamente a la base de datos MySQL. En caso contrario, la información se almacena temporalmente en la EEPROM.

3. Recuperación y Sincronización

Cuando se restablece la conexión WiFi, el sistema recupera los registros almacenados en la EEPROM y los sincroniza con la base de datos en orden cronológico. Una vez enviados correctamente, los datos se eliminan de la EEPROM para liberar espacio.

4. Base de Datos MySQL

La base de datos está alojada localmente en un servidor XAMPP. Fue modificada para incluir la columna humedad_dht11, lo que permite almacenar la información proveniente de ambos sensores de humedad de forma separada.
