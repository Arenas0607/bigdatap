RESUMEN

Este proyecto tiene como objetivo el diseño y desarrollo de un sistema de interés basado en el microcontrolador ESP32, capaz de capturar datos ambientales desde sensores (DHT11, DHT22, MQ8), almacenar dicha información en una base de datos MySQL utilizando una conexión Wifi, y garantizar la continuidad del almacenamiento mediante una EEPROM AT24C32 en caso de fallos en la conectividad. Esta solución está pensada para aplicaciones donde la conectividad no es constante, permitiendo que los datos no se pierdan y se sincronicen posteriormente de manera automática.
