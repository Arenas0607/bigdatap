Componentes

Principales Dispositivos y Módulos

  • ESP32: Microcontrolador principal con conectividad WiFi.
  • DHT11: Sensor de temperatura y humedad de baja precisión.
  • DHT22: Sensor de temperatura y humedad de alta precisión.
  • MQ8: Sensor de gas hidrógeno.
  • DS3231: Módulo RTC para mantener la fecha y hora con alta precisión.
  • EEPROM AT24C32: Memoria I2C utilizada para el almacenamiento temporal de datos.
  • Base de Datos MySQL: Sistema de almacenamiento implementado en un servidor local (XAMPP).

Conexiones de Hardware

  • MQ8: Pin D34
  • DHT22: Pin D4
  • DHT11: Pin D23
  • RTC DS3231 (I2C): D21 (SDA), D22 (SCL)
